You found it

So glad that you could join us. This is where you can see some of the projects I've worked on.

I mostly use HTML, CSS, and Javascript to manipulate pixels on screens in an attempt to solve problems. Sometimes that problem is just my own curiosity. Have a look around and thanks for stopping by.

Jonathan Fox Poetry
This is a Wordpress site I built for a poet. It showcases some of his recent work. I designed and coded the site for him, made the logo and helped him to upload the content. I tried to make it a pleasant reading experience.

Hot Pink Lazers
This is a version of the game 'Asteroids'. You can play it with a keyboard or with a controller. This is the first game I made that I can play from my couch with a controller. I doubt it will be the last.

War Is A Force That Gives Us Meaning
This is a simple shooting game built using Javascript. It's playable on a touch screen or with a keyboard. I really haven't modified this game much from the original tutorial, but it is a multi state game with a boss a and a win screen.
